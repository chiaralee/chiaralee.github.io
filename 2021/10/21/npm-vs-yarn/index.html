查看源和换源

描述 npm yarn
查看当前镜像源 npm config get registry yarn config get registry
设置镜像源 npm config set registry https://registry.npm.taobao.org/ yarn config set registry https://registry.npm.taobao.org/

还可以直接打开npm 配置文件设置源

依赖管理

描述 npm yarn
项目安装所有依赖 npm [install/i] yarn
项目卸载所有依赖 npm [uninstall/un] yarn remove
全局安装 npm install [–-global/-g] yarn global add
安装依赖 npm [install/i] [—save/-S] yarn add
开发模式安装依赖 npm [install/i] [—save-dev/-D] yarn add [—dev/-D]
卸载依赖 npm [uninstall/un] yarn remove
更新依赖 npm update —save yarn upgrade
全局更新 npm update –-global yarn global upgrade
清除缓存 npm cache clean yarn cache clean
